# paper-insight

a website for analyzing academic papers, whether for the authors or for those interested in better understanding the work. He will help with writing certain parts and also explain. The idea is a very simple and practical saas. The main page will only have a header, a dotted box to drag & drop the PDF and selectboxes with actions

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/paper-insight.git
cd paper-insight
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
